# Marketing UI Kit — Claude Code

A click-thru recreation of the Claude Code marketing landing page (`claude.com/product/claude-code`).

## Components

- `Nav.jsx` — top nav with wordmark and primary CTA
- `Hero.jsx` — editorial hero with terminal demo
- `TerminalWindow.jsx` — reusable terminal frame (used in hero + feature blocks)
- `FeatureGrid.jsx` — 3-up feature cards
- `SurfacesSection.jsx` — Terminal / VS Code / Desktop / Web surfaces tabs
- `CTABand.jsx` — closing call-to-action
- `Footer.jsx` — minimal footer

## How to view

Open `index.html` directly. It loads React + Babel from CDN and the components from this folder.
